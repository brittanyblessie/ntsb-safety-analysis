# NTSB Aviation Safety Analysis

Exploratory analysis of US civil aviation accident and fatality trends 
using the NTSB accident database (1982-present).

## What this project covers

### Civil Aviation Trends
- Accidents by year (1982-2024)
- Top 10 accident types by phase of flight
- Fatalities by year

### Helicopter Analysis
- Top 10 helicopter accident types
- Helicopter accident trends by year (1982-2024)
- Exploration of rotorcraft risk patterns as context for 
  emerging eVTOL operations

## Tools used
- Python
- Pandas
- Matplotlib
- pyodbc

## Data source
NTSB Aviation Accident Database
https://www.ntsb.gov/safety/data/Pages/Data_Stats.aspx

## About
This project was built to apply data science skills to aviation safety 
data. The helicopter analysis looks at historical rotorcraft accident 
patterns as a reference point for understanding the safety challenges 
facing new categories of electric aircraft.
